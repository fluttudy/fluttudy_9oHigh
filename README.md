# Fluttudy_9oHigh
* Flutter랑 친해지기

## 작성 규칙
* 폴더와 파일명은 영문으로 작성하기
  * 폴더명 : Camel case로 작성
  * 파일명 : Snake case로 파일명 작성

* 모든 내용은 제목과 구분점으로만 구분해 작성하기
